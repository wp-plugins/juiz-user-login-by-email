== Installation ==

You can use one of both method :

**Installation via your Wordpress website**

1. Go to the **admin menu 'Plugins' -> 'Install'** and **search** for 'User login by email'
1. **Click** 'install' and **activate it**
1. That's all for you!

**Manual Installation**

1. **Download** the plugin (it's better :p)
1. **Unzip** `juiz-user-login-by-email` folder to the `/wp-content/plugins/` directory
1. Go to your WP admin panel to "Appearance > Plugins" and activate
1. That's all for you!
